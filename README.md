# StackFoundry

StackFoundry is a modern technical blog platform built using **HTML, CSS, JavaScript, and React (via CDN)**.  
It provides a clean reading experience for long-form engineering articles covering topics like systems programming, web architecture, databases, and AI/ML.

This project demonstrates a full blog-style frontend with features like article feeds, comments, author pages, bookmarks, and a writing editor.

# Features

## Core Features
- Article feed with search and filtering
- Category and tag-based navigation
- Author profile pages
- Article reading interface with syntax-highlighted code blocks
- Table of contents for articles
- Reading progress bar
- Comment system
- Like and bookmark functionality

## User Features
- Login / signup UI
- Write and publish articles
- Save articles for later
- Dark mode toggle

## UI / UX
- Responsive layout
- Clean typography optimized for long-form reading
- Trending articles sidebar
- Newsletter subscription box
- Related article recommendations
